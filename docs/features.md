# Features

- Completion: context-aware completions from API index, XML libraries, and workspace symbols.
- Hover: preview snippets and documentation when available.
- Go To Definition: jump to symbols in workspace and XML-backed libraries.
- Rename: workspace-aware rename with heuristic filtering.
- Extract Method: preview and apply refactor with inferred parameters.
- Persistent index: incremental on-disk index for faster startup.
